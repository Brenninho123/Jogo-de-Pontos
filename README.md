# Jogo de Pontos

Um jogo simples de pontuação para navegador, usando HTML, JavaScript e opcionalmente C++ via WebAssembly.

## Funcionalidades

- Adicionar pontos clicando em um botão
- Resetar pontuação
- [Opcional] Usar lógica C++ compilada para WebAssembly

## Como rodar

1. Abra o arquivo `index.html` no navegador
2. Clique em “Adicionar Ponto” para aumentar a pontuação
3. Clique em “Resetar” para zerar a pontuação

## Tecnologias

- HTML5
- CSS3
- JavaScript
- C++ (opcional via WebAssembly)
